# **App Name**: Kevin Spotter

## Core Features:

- Kevin Login: Option to log in as 'soy kevin'.
- Non-Kevin User Creation: Allow users who are 'no soy kevin' to create a username (no password or email required).
- Photo Upload: Enable 'soy kevin' users to upload photos.
- Location Submission: Enable 'soy kevin' users to submit their current location.
- Comment Submission: Allow 'soy kevin' users to add a comment to their post.
- Location API Integration: Fetch location data using an external API.
- Data Display: Display photos, locations, and comments from 'soy kevin' to 'no soy kevin' users. Using supabase for persistence.

## Style Guidelines:

- Primary color: Warm orange (#FF7F50) to evoke a sense of comfort and friendliness.
- Background color: Soft peach (#FAEBD7) to create a welcoming, comfy atmosphere.
- Accent color: Pale yellow (#FAFAD2) for highlights and interactive elements, providing a gentle contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif, for readability and a touch of warmth.
- Mobile-first responsive design ensuring optimal viewing experience across various devices.
- Use simple, clear icons to represent actions and data points, enhancing usability.
- Subtle transitions and animations to provide feedback and guide the user through the app, creating a smooth and engaging experience.
